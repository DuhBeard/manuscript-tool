# manuscript-tool

This is a tool designed to assist with sermon and teaching manuscript preparation by identifying, verifying, and highlighting Scripture references in documents.

## Features

- Detects Bible references in text documents (e.g., sermons, teaching notes)
- Retrieves verse text using the API.Bible service
- Highlights Scripture passages for easy review
- (Planned) Compare manuscript wording with official Bible text for accuracy
- (Planned) Support multiple Bible translations

## Purpose

This project is intended for:
- Church leaders
- Teachers
- Students preparing Bible-based content

It is designed for **study and preparation use only**, not for redistributing Bible text as a standalone reading application.

## Status

In development

## Tech Stack

- Python (document parsing + processing)
- API.Bible (Scripture retrieval)
- python-docx (Word document handling)

## Notes

This project uses API.Bible to access Scripture text and complies with their usage guidelines.
